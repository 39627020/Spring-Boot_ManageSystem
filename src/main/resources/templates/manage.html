CPS订阅端后台API

API概述

提供订阅端API服务，包括POST订阅请求，GET历史订阅信息，DELETE删除订阅信息。 API风格是RestFul，不需要身份验证

API域名及端口

115.158.145.201:8080/
编号 API名字 API功能 API 请求方法 详情
1 查询订阅信息 查询历史订阅信息 cps/api/v1/subInfos GET 点击查看详情
2 提交订阅信息 提交本次订阅信息 cps/api/v1/subInfos POST 点击查看详情
3 删除制定信息 删除制定信息 cps/api/v1/subInfos/{id} DELETE 点击查看详情
